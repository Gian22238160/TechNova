Descripción del producto

La Placa Madre MSI B550M PRO está diseñada para ofrecer rendimiento y estabilidad en equipos de alto desempeño. Compatible con procesadores AMD Ryzen™ con socket AM4, combina potencia, eficiencia y conectividad moderna.

Con soporte para memorias DDR4 de alta velocidad, ranuras PCIe 4.0 para tarjetas gráficas de última generación y almacenamiento M.2 NVMe ultrarrápido, esta placa es ideal para gaming, trabajo de oficina o creación de contenido.

Además, su diseño PRO Series garantiza durabilidad, excelente disipación térmica y una interfaz BIOS intuitiva para aprovechar todo el potencial de tu sistema.

Especificaciones técnicas

  • Socket: AMD AM4
  • Chipset: AMD B550
  • Formato: Micro-ATX
  • Memoria: 4x DDR4 (hasta 128GB, 4400+ MHz OC)
  • Gráficos: 1x PCIe 4.0 x16, 1x PCIe 3.0 x1
  • Almacenamiento: 1x M.2 NVMe PCIe 4.0, 4x SATA III
  • Audio: Realtek ALC892/897 7.1 canales
  • Red: LAN Gigabit Realtek
  • Puertos traseros: HDMI, DVI-D, USB 3.2 Gen1/Gen2, RJ45, Audio
  • Alimentación: 24-pin ATX + 8-pin CPU
  • Compatibilidad: AMD Ryzen 3000 / 4000G / 5000 Series
